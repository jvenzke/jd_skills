# General skills 
- `\grill-me`: get the agent and user on the same page about a problem/project
- `\caveman`: reduces token usage


# Research agent workflow 
- `\generate-research-project`: takes jira task and generates git branch and folder for work  
- `\research-plan`: takes an idea and produces a fleshed out research plan 
- Research loop (repeat till complete)
     - `\research-to-tasks`: Takes the research plan and makes the next few tasks to do
     - `\do-research`: does the research work for a single task
     - `\review-research`: reviews work from last task and suggests if more research is needed or not
- `\summarize-research`: Summarizes the results of all the work to prepare it to move to prod

# Development 
- `\grill-me`: get the agent and user on the same page about a problem/project
- `\to-prd`: develops a prd based on the discussion from `\grill-me`
- `\prd-to-issue-files`: creates issues for each step of the PRD
